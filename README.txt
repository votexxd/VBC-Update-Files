How to Update:

1. Move all the Files from the UpdateFiles folder to "...\Documents\VBC-Backgrounds"
2. Delete the old "Votex's Background Changer v[version].exe" in "...\Documents\VBC-Backgrounds" and the shortcut on your Desktop

How to create a new Shortcut:

1. Rightclick the new "Votex's Background Changer v[version].exe" -> Send to -> Desktop
